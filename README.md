## ğŸ“¦ Installation

### System Requirements

#### Desktop Application
- **OS**: Windows 10/11, macOS 10.15+, Linux
- **RAM**: 4GB (8GB recommended)
- **Storage**: 500MB free space
- **Java**: JDK 11 or higher

#### Web Application
- **Node.js**: 14.x or higher
- **npm**: 6.x or higher
- **MySQL**: 8.0 or higher
- **Browser**: Chrome, Firefox, Safari, Edge (latest versions)

### Prerequisites
- Java Development Kit (JDK) 11 or higher
- Node.js and npm
- MySQL Server 8.0+
- Git
- Maven 3.6+

### Quick Start

#### Option 1: Using Docker (Recommended)
```bash
# Clone the repository
git clone https://github.com/Keshabkjha/travel-booking-planner.git
cd travel-booking-planner

# Start the application
docker-compose up -d
```

#### Option 2: Manual Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/Keshabkjha/travel-booking-planner.git
   cd travel-booking-planner
   ```

2. **Set up the backend**
   ```bash
   cd "travel-and-booking planner website"
   mvn clean install
   ```

3. **Set up the frontend**
   ```bash
   cd ../travel-planner-frontend
   npm install
   npm run build
   ```

4. **Configure environment variables**
   Create a `.env` file in the root directory with the following variables:
   ```env
   DB_URL=jdbc:mysql://localhost:3306/travel_planner
   DB_USERNAME=your_username
   DB_PASSWORD=your_password
   JWT_SECRET=your_jwt_secret
   ```

5. **Start the application**
   ```bash
   # Start backend
   mvn spring-boot:run
   
   # In a new terminal, start frontend
   npm start
   ```

The application will be available at `http://localhost:3000`

### Prerequisites
- Java Development Kit (JDK) 8 or higher
- Apache Maven
- MySQL Server
- Apache Tomcat (for web application)
- Git

### Desktop Application
```bash
# Clone the repository

# Deploy the WAR file to your Tomcat server
cp target/travel-planner.war $TOMCAT_HOME/webapps/
```

## ğŸ’» Usage

### Getting Started
1. **Create an Account**
   - Click on "Sign Up" and fill in your details
   - Verify your email address
   - Complete your profile setup

2. **Plan Your Trip**
   - Click on "New Trip"
   - Enter your destination and travel dates
   - Add flights, hotels, and activities
   - Invite travel companions (web version only)

3. **Manage Bookings**
   - View all your bookings in one place
   - Make changes or cancellations
   - Download receipts and itineraries

### Advanced Features
- **Expense Tracker**: Log and categorize your travel expenses
- **Packing List**: Create and share packing lists
- **Travel Documents**: Store important documents securely
- **Offline Mode**: Access your plans without internet (desktop only)

### Keyboard Shortcuts (Desktop App)
| Shortcut | Action |
|-----------|--------|
| `Ctrl + N` | New Trip |
| `Ctrl + F` | Search |
| `Ctrl + S` | Save |
| `Ctrl + P` | Print |
| `F1` | Help |

### Desktop Application
1. Launch the application
2. Create a new account or log in
3. Start planning your trip by adding destinations
4. Book flights and hotels through the integrated booking system
5. Manage your itinerary and expenses

### Web Application
1. Access the web application through your browser
2. Sign up or log in to your account
3. Sync with your desktop application (if needed)
4. Access your travel plans from anywhere
5. Share your itinerary with travel companions
